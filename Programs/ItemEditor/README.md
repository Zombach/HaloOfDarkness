## Задача: Разработка приложения "Предмет Эдитор" на WPF/UWP

### Описание
Требуется разработать приложение с интерфейсом на
основе WPF (Windows Presentation Foundation) или UWP
(Universal Windows Platform), считывающее и модифицирующее
базу предметов. Приложение должно иметь настройки, где
можно указать путь к базе предметов на диске или в базе
данных SQL. Если база отсутствует, приложение должно
позволить выбрать новый путь.

### Детали задачи
- Разработать концепцию, считывающую и модифицирующую
  базу предметов.
- Поддержка выбора предметов по идентификатору.
- Загруженные предметы имеют шаблон, зависящий от типа:
	1. Идентификатор
	2. Имя
	3. Тип
	4. Вес
	5. Цена
- Приложение должно предоставлять возможность создания,
  редактирования и удаления предметов.

### Функциональные требования
1. Загрузка, редактирование и удаление предметов.
2. Выбор предметов по идентификатору.
3. Поддержка настроек с указанием пути к базе предметов.

### Примеры использования
1. Открытие базы предметов и выбор предмета по идентификатору.
2. Редактирование характеристик предмета.
3. Создание нового предмета.
4. Удаление предмета из базы.

### Замечания
- Подробная инструкция по запуску и использованию.
- Возможность работы с базой данных SQL.
- Удобный интерфейс для редактирования предметов.
