## Задача: Разработка утилиты упаковки и распаковки данных

### Описание
Требуется разработать консольное приложение на языке, предназначенное для
упаковки и распаковки файлов в архив. Программа должна иметь возможность работы как
с отдельными файлами, так и с каталогами, а также обладать функциональностью
фильтрации файлов при упаковке каталогов. При упаковке файлов, программе
необходимо создавать архивный файл с соответствующим именем и индексом. Внутри
архива должен располагаться файл описания, который содержит информацию о местоположении
файлов и их описании.

### Требования
1. Разработать функциональность упаковки и распаковки файлов и каталогов.
2. Обеспечить возможность указания фильтра типов файлов при упаковке каталога.
3. Создавать архивный файл (или несколько с индексами) с общим именем.
4. Внутри архива должен быть файл описания, содержащий информацию о содержимом и
расположении файлов.
5. Предоставить поддержку составных файлов и указание всех частей.

### Файл описания
Файл описания, находящийся внутри архива, представляет собой текстовый файл, который
содержит следующую информацию:
- Список файлов в архиве и их местоположение внутри архива.
- Описание каждого файла (например, имя, размер, дата создания).
- Информацию о размещении составных файлов и их частях, если таковые присутствуют.

### Примеры
1. Упаковка одного файла: `PackUnpack.exe --pack file.txt`
2. Упаковка каталога с фильтром: `PackUnpack.exe --pack directory/ --filter *.xml`
3. Распаковка архива: `PackUnpack.exe --unpack archive.pack`

### Замечания
- Предоставьте подробную инструкцию по запуску и установке необходимых зависимостей
для использования приложения.
- Рассмотрите возможность передачи параметров упаковки и распаковки через аргументы
командной строки, например, с использованием библиотеки `System.CommandLine`.
- Фильтр типов файлов может быть задан через опциональный аргумент, например,
`--filter`, с указанием шаблона фильтрации.
