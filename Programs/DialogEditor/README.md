## Задача: Разработка приложения "Диалог Эдитор" на WPF/UWP

### Описание
Требуется разработать приложение с интерфейсом на
основе WPF (Windows Presentation Foundation) или UWP
(Universal Windows Platform), считывающее и модифицирующее
базу диалогов. Приложение должно иметь настройки, где
можно указать путь к базе диалогов на диске или в базе
данных SQL. Если база отсутствует, приложение должно
позволить выбрать новый путь.

### Детали задачи
- Разработать концепцию, считывающую и модифицирующую
  диалоги.
- Поддержка выбора диалогов по идентификатору.
- Открытое окно: выбор идентификатора объекта для диалога.
- Диалоги с ветвистыми деревьями и вариантами ответов.
- Редактирование текста диалога с древовидной структурой.
- Поддержка предметов, параметров и логических проверок.
- Ограничение доступности веток на основе условий.

### Функциональные требования
1. Создание и редактирование диалогов.
2. Выбор диалога по идентификатору.
3. Привязка диалога к объекту.
4. Редактирование текста диалогов.
5. Добавление предметов, параметров и элементов.
6. Ограничение доступности веток на основе условий.

### Примеры использования
1. Открытие базы диалогов и выбор по идентификатору.
2. Редактирование текста диалога с вариантами ответов.
3. Привязка диалога к объекту в игре.
4. Добавление элементов в диалог для проверок.
5. Ограничение доступности веток на основе условий.

### Замечания
- Подробная инструкция по запуску и использованию.
- Возможность работы с базой данных SQL.
- Удобный интерфейс для редактирования диалогов.
