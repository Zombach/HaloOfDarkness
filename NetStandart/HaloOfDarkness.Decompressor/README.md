## Библиотека "Decompressor"

### Описание
Библиотека `Decompressor` предназначена для работы с запакованными данными, обеспечивая функциональность по распаковке архивов. Она поддерживает как одиночные архивы, так и многокомпонентные архивы.

### Функциональность
1. **Открытие архива:** `Decompressor` позволяет открыть `Stream` данных и перейти к файлу, в котором описаны вложенные файлы данного архива.

2. **Поиск файла:** При наличии архива, `Decompressor` осуществляет поиск запрошенного файла внутри него.

3. **Управление многокомпонентными архивами:** Если файл находится в другой части архива, `Decompressor` корректно переоткрывает `Stream` и переходит к соответствующей части архива.

4. **Извлечение файла:** `Decompressor` извлекает файл из архива и выгружает его. Если файл охватывает несколько частей архива, `Decompressor` объединяет их корректно.

5. **Использование Span<byte>:** Выгрузка данных из архива осуществляется с использованием `Span<byte>`, что обеспечивает эффективное управление памятью.

6. **Оптимизированная проверка:** При сравнении байтов с ожидаемыми данными для выявления начала следующей части файла, `Decompressor` выполняет оптимизированную проверку, с целью минимизации использования ресурсов.

### Примеры использования
1. Открытие и распаковка одиночного архива.
2. Обработка многокомпонентного архива с необходимостью переоткрытия `Stream`.
3. Извлечение файла из архива и правильное объединение данных из нескольких частей.
4. Эффективная выгрузка данных с использованием `Span<byte>`.
5. Проверка корректности распакованных данных с оптимизированным сравнением.

### Замечания
- Библиотека `Decompressor` обеспечивает эффективную и надежную работу с архивами, при этом минимизируется использование ресурсов.
- Правильная обработка многокомпонентных архивов и корректное объединение данных являются ключевыми аспектами функциональности библиотеки.
- Оптимизированная проверка сравнением байтов обеспечивает быструю и точную верификацию данных.
