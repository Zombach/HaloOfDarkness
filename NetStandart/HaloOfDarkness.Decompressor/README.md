		<h1 style="text-align: center">Decompressor</h1>
		<sector style="text-align: justify">
			<p>Decompressor - библиотека предназначенная для работы с запакованными данными</p>
			<br/>
			<p>Архив может быть, как одиночный, так и разделенный на части.</p>
			<p>Decompressor должна открывать Stream  и переходить к файлу в котором описаны вложенные файлы данного пака</p>
			<p>Далее проверяет наличие запрошенного файла</p>
			<p>Если файл находится в другой его части, то переоткрывает стрим</p>
			<p>Переходит к файлу и выгружает его</p>
			<p>Если же файл попал на границы частей, то необходимо выгрузить две части и правильно их соеденить</p>
			<p>Фактически выгрузка должна происходить через Span<"byte"></p>
			<p>А сама проверка, через сравнения байта'ов в ожидаемыми данными, чтобы не тратить лишние ресурсы</p>
